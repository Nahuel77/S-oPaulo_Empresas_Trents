Proyecto: SP-Empresas-Trends

SP-Empresas-Trends es una aplicación full-stack destinada a recolectar, procesar y analizar datos públicos sobre la apertura de nuevas empresas en la ciudad de São Paulo. El sistema combina:

Scraping / ingestión automática de datos públicos (registros de empresas, actividad económica, geolocalización).

Procesamiento y limpieza de datos (normalización, georreferenciación, categorización por CNAE/sector).

Análisis de tendencias y clusterización de sectores emergentes.

Backend en Python, con API para servir datos procesados.

Frontend moderno (dashboard) para visualizar estadísticas: cantidad de empresas por barrio, sectores más activos, evolución temporal, mapas de densidad, gráficos dinámicos.

Optional: uso de modelos NLP / Transformers para generar resúmenes automáticos de insights, informes ejecutivos o alertas de cambios.

Objetivo: ofrecer una vista actualizada, navegable y analizable del panorama empresarial de São Paulo, ideal para análisis de mercado, estudios económicos, emprendimientos, inversionistas o políticas urbanas.
